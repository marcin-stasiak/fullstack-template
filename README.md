# Fullstack Template

## Description

Fullstack template for Nest & React web applications

## Project setup

```bash
# copy and edit .env file
$ cp .env.dist .env
```

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
